PyTom: About

What PyTom does:

PyTom (Python and Tomography) is a toolbox dedicated for processing cryo electron tomograms. (Hrabe et. al. 2012)

What PyTom does:

PyTom covers key functions for processing three-dimensional data, such as cryo electron tomograms.

It supports basic reconstruction procedures (e.g., weighted backprojection), algorithms for detecting macromolecules in tomograms (e.g., template matching), and subtomogram analysis (e.g., subtomogram averaging and classification.

(Hrabe, Foerster 2011)
